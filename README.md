# yunting

在欧卡/美卡中收听云听电台

插件版使用说明：将`yunting_plugin.dll`放入`游戏文件夹\bin\win_x64\plugins`下（若没有 plugins 文件夹可以创建一个），自动为游戏生成 live_streams.sii

程序版使用说明：直接运行`yunting.exe`，会在当前目录生成 live_streams.sii

## 配置文件

程序运行后会在当前文件夹，插件运行后会在文档文件夹生成`yunting.toml`配置文件，可以配置需要包含的的省市列表
